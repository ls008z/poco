# poco

> ⏱️ Built in under 4 hours with [ChatGPT](https://openai.com/chatgpt)  
> 💡 Collaborative coding + design + debugging + plotting — all in one session

**POwer and COverage (poco)** is a Python framework for simulation-based evaluation of statistical estimators. It helps you generate power curves, coverage curves, bias diagnostics, and more — using empirical or parametric data, with support for full parallelization and interactive visualization.

---

## 🚀 Features

- 📦 **Modular API** for estimators, generators, and simulation workflows
- 🔁 **Empirical + Parametric generators** (bootstrap, normal, etc.)
- ⚡ **Parallel processing** for embarrassingly parallel simulation tasks
- 📈 **Plotting tools** for:
  - Power curves
  - Coverage curves
  - Estimate + SE distributions
  - Bias and SE bias curves
- 🧪 Reproducible, deterministic, seed-driven design
